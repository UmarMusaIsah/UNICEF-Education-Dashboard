# UNICEF Education Dashboard

## Project Overview

This project is a UNICEF-inspired education analytics dashboard developed using Excel and Power BI. It analyzes key education indicators such as student enrollment, attendance, gender parity, teacher availability and school infrastructure across Kano State.

## Objectives

* Track school enrollment trends
* Measure attendance performance
* Compare male vs female participation
* Identify teacher shortages
* Highlight infrastructure gaps

## Tools Used

* Microsoft Excel
* Power BI
* Power Query
* Data Cleaning Techniques

## Dashboard Features

* KPI cards
* Enrollment trends by LGA
* Gender parity analysis
* Attendance comparison
* Teacher-to-student ratio
* Interactive filters

## Key Insights

* Rural LGAs showed lower female attendance.
* Some zones had higher classroom shortages.
* Enrollment improved in urban areas.
